# 0x04. C - More functions, more nested loops

## Description
This project is about learning more about C functions and nested loops. We are also introduced to header files, variable scope, and using standard restricted C environments.

## Files
| File | Description |
| --- | --- |
| `0-isupper.c` | Checks for uppercase character. |
| `1-isdigit.c` | Checks for a digit (0 through 9). |
| `2-mul.c` | Multiplies two integers. |
| `3-print_numbers.c` | Prints the numbers, from 0 to 9. |
| `4-print_most_numbers.c` | Prints the numbers, from 0 to 9, except 2 and 4. |
| `5-more_numbers.c` | Prints 10 times the numbers, from 0 to 14. |
| `6-print_line.c` | Draws a straight line in the terminal. |
| `7-print_diagonal.c` | Draws a diagonal line on the terminal. |
| `8-print_square.c` | Prints a square. |
| `9-fizz_buzz.c` | Prints the numbers from 1 to 100 (FizzBuzz). |
| `10-print_triangle.c` | Prints a triangle. |
| `main.h` | Header file containing all function prototypes. |
| `_putchar.c` | Custom _putchar function to print characters. |
