# C - argc, argv

## Description
This project is part of the low-level programming curriculum. It focuses on understanding how to pass, read, and handle command-line arguments in C programs using `argc` (argument count) and `argv` (argument vector).

## Learning Objectives
By the end of this project, we are expected to be able to explain:
* How to use arguments passed to a program.
* What are two prototypes of `main` that we know of, and in which case to use one or the other.
* How to use `__attribute__((unused))` or `(void)` to compile functions with unused variables or parameters.

## Requirements
* Operating System: Ubuntu 20.04 LTS
* Compiler: `gcc` with options `-Wall -Werror -Wextra -pedantic -std=gnu89`
* Style guidelines: Betty style (`betty-style.pl` and `betty-doc.pl`)
* Editors: `vi`, `vim`, `emacs`
* No global variables are allowed.
* No more than 5 functions per file.
* All files should end with a new line.
